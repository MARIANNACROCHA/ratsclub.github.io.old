Setup declarativo de e-mail utilizando Nix e notmuch

2020-08-06
#emacs  #nix  #email 
Sempre utilizei o Thunderbird como cliente de e-mail, entretanto uma coisa sempre me incomodou: a busca é lenta. Isso se mostra verdadeiro até mesmo para uma pessoa que faz um uso leve de e-mails. Aqui entra o notmuch, de acordo com seu próprio site, é um sistema de busca e tags globais para e-mails. Ele é encarregado por fazer buscas rápidas (o site garante que milhões de mensagens não são muito) e manter sua caixa de entrada vazia através de seu sistema de tags.
